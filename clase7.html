Atención especial

Conceptos que suelen confundirse

Diferencias importantes que es muy probable que aparezcan en el parcial. Prestá especial atención a estas comparaciones.

Confusión frecuente 1

Dato vs. Información

No son lo mismo:

  • Dato: valor crudo, sin contexto. Ej: "25"
  • Información: dato procesado con contexto y significado. Ej: "La temperatura hoy es 25°C"

La informática se encarga de transformar datos en información útil para el usuario.

Confusión frecuente 2

Hardware vs. Software

Son las dos partes de cualquier sistema informático:

  • Hardware: componentes físicos. Se puede tocar. Ej: teclado, disco rígido, CPU.
  • Software: programas e instrucciones. No se puede tocar. Ej: Windows, Word, Chrome.

Sin hardware el software no puede ejecutarse; sin software el hardware no hace nada útil.

Confusión frecuente 3

Sistema Operativo vs. Aplicación

Dos tipos de software con roles muy distintos:

  • Sistema operativo: gestiona el hardware y es la base sobre la que corren los demás programas. Ej: Windows, Linux, macOS.
  • Aplicación: programa que usa el SO para realizar una tarea específica. Ej: Word, Chrome, Spotify.

El SO es el intermediario entre el hardware y las aplicaciones.

Confusión frecuente 4

LAN vs. WAN

Tipos de red clasificados por su alcance geográfico:

  • LAN (Local Area Network): red de área local. Ej: la red de una escuela o una oficina.
  • WAN (Wide Area Network): red de área amplia. Cubre países o continentes. Ej: Internet.

Existen también MAN (ciudad) y PAN (personal, muy corto alcance como Bluetooth).

Confusión frecuente 5

Internet vs. World Wide Web

Son conceptos relacionados pero distintos:

  • Internet: la infraestructura física y lógica de redes interconectadas a nivel mundial.
  • World Wide Web (WWW): uno de los servicios que funciona sobre Internet: el sistema de páginas web enlazadas.

El email, el FTP y el chat también usan Internet pero no son parte de la Web.

Confusión frecuente 6

Navegador vs. Buscador

Es uno de los errores más comunes:

  • Navegador: programa instalado en tu dispositivo que te permite acceder a Internet. Ej: Chrome, Firefox, Edge.
  • Buscador: sitio web al que accedés desde el navegador para encontrar información. Ej: Google, Bing, DuckDuckGo.

Usás Chrome (navegador) para entrar a Google (buscador).

Confusión frecuente 7

Página web vs. Sitio web

Dos niveles de la misma estructura:

  • Página web: un único documento HTML accesible por una URL. Ej: la página de inicio de Wikipedia.
  • Sitio web: conjunto de páginas web relacionadas bajo el mismo dominio. Ej: todo Wikipedia.

Un sitio web contiene muchas páginas web enlazadas entre sí.

Confusión frecuente 8

RAM vs. ROM

Dos tipos de memoria con funciones opuestas:

  • RAM (Random Access Memory): memoria temporal, se borra al apagar. Almacena lo que está en uso ahora.
  • ROM (Read Only Memory): memoria permanente, solo lectura. Contiene instrucciones básicas del sistema (BIOS).

Cuanta más RAM tiene un equipo, más programas puede tener abiertos al mismo tiempo sin volverse lento.

Modelo orientativo

Modelo de Parcial

Este es un modelo de referencia: las preguntas reales del examen pueden variar. El parcial puede incluir preguntas de desarrollo, múltiple choice o una combinación de ambos formatos.

Modelo · Parte 1

Unidad 1: Informática y sistemas

  1. Definí informática e indicá cuál es la diferencia entre un dato y la información. Dá un ejemplo de cada uno.

  2. Explicá la diferencia entre hardware y software. Mencioná al menos 3 ejemplos de cada uno.

  3. ¿Qué es un sistema operativo? ¿Cuáles son sus funciones principales? Nombrá 3 sistemas operativos de escritorio y 2 de dispositivos móviles.

  4. Describí los siguientes componentes de una computadora e indicá si son hardware o software: CPU, RAM, disco rígido, sistema operativo, teclado.

  5. ¿Qué es un archivo? Explicá para qué sirve la extensión de un archivo y mencioná 5 extensiones comunes indicando el tipo de contenido que representan.

Modelo · Parte 2

Unidades 1 y 2: Redes y Web

  1. Definí red informática y explicá la diferencia entre una red LAN y una red WAN. Dá un ejemplo real de cada una.

  2. Nombrá y describí brevemente 3 dispositivos de red (switch, router, módem, etc.) e indicá cuál es la función de cada uno.

  3. ¿Qué es Internet? ¿Cuál es la diferencia entre Internet y la World Wide Web? Mencioná 4 servicios que ofrece Internet.

  4. Explicá qué es un buscador web, cómo funciona y cuál es la diferencia con un navegador. Dá un ejemplo de cada uno.

  5. ¿Qué es una página web? ¿En qué lenguaje están escritas? Explicá brevemente cómo un navegador interpreta ese lenguaje para mostrarte la página.
