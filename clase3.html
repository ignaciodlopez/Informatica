Manejo de Archivos y Carpetas

Una guía visual para comprender cómo se organizan los archivos, qué son las carpetas, cómo usar el Explorador de Windows y cómo gestionar la información de manera eficiente y ordenada.

Objetivo de la clase

Gestionar archivos y carpetas de forma eficiente

Identificar qué es un archivo, cómo se organizan en carpetas, cómo utilizar el Explorador de Windows y realizar operaciones básicas para mantener la información ordenada y accesible.

Crear Organizar Gestionar
Enfoque de la clase

Saber organizar archivos no es sólo una cuestión técnica: es la base para trabajar de manera productiva. Una buena estructura de carpetas ahorra tiempo, evita pérdidas de información y facilita el trabajo colaborativo y el acceso rápido a lo que se necesita.

Concepto fundamental

¿Qué es un archivo?

Un archivo es un conjunto de datos o información almacenados en la computadora que pueden ser identificados por un nombre único. Los archivos son la unidad básica de almacenamiento y pueden contener texto, imágenes, sonido, video, programas o cualquier otro tipo de información digital.

Nombre del archivo

Cada archivo tiene un nombre que lo identifica. Debe ser único dentro de la misma carpeta y puede contener letras, números y algunos caracteres especiales.

Ejemplo: "Tarea Final.docx", "Presupuesto 2026.xlsx", "foto_vacaciones.jpg"

Extensión del archivo

Es un sufijo de 3 o 4 letras que aparece después del punto en el nombre del archivo. Indica el tipo de archivo y qué programa se necesita para abrirlo.

Ejemplos: .docx (Word), .pdf (documento portátil), .jpg (imagen), .mp3 (audio), .mp4 (video)

Propiedades del archivo

Cada archivo tiene propiedades asociadas: tamaño, fecha de creación, fecha de modificación, tipo y atributos de solo lectura u oculto.

Ejemplo: Un documento de 250 KB creado el 10/04/2026 y modificado el 15/04/2026

Tipos comunes

Archivos de documentos

Contienen texto, tablas, presentaciones y otro contenido de oficina. Son los más utilizados en entornos educativos y laborales.

  • .docx / .doc: Documentos de Microsoft Word
  • .xlsx / .xls: Planillas de cálculo de Excel
  • .pptx / .ppt: Presentaciones de PowerPoint
  • .pdf: Formato de documento portátil
  • .txt: Archivos de texto plano

Tipos comunes

Archivos multimedia

Almacenan contenido audiovisual: fotografías, música, videos y gráficos que enriquecen la comunicación y el entretenimiento.

  • .jpg / .jpeg / .png: Imágenes y fotografías
  • .mp3 / .wav: Archivos de audio
  • .mp4 / .avi / .mkv: Archivos de video
  • .gif: Imágenes animadas

Organización

Carpetas: los contenedores de archivos

Las carpetas (también llamadas directorios) son contenedores lógicos que permiten organizar archivos y otras carpetas de forma jerárquica. Funcionan como los cajones de un escritorio físico: agrupan elementos relacionados para facilitar su búsqueda y acceso.

Concepto

¿Qué es una carpeta?

Es un espacio de almacenamiento que puede contener archivos y otras carpetas (subcarpetas). Permite crear una estructura organizada de la información.

  • Agrupa archivos relacionados
  • Puede contener otras carpetas
  • Facilita la navegación y búsqueda

Ejemplo: Mis Documentos → Facultad → Informática → Clase 3

Organización

Carpetas del sistema

Windows crea carpetas especiales al instalar el sistema: Documentos, Descargas, Imágenes, Videos, Música y Escritorio. Son puntos de partida para organizar archivos personales. Las carpetas se organizan en forma de árbol jerárquico.

  • Mis Documentos: archivos de trabajo y estudio
  • Descargas: archivos descargados de internet
  • Escritorio: acceso rápido a archivos frecuentes
  • Imágenes: fotos y gráficos
  • Videos: archivos multimedia
Ventajas de una buena organización

Mantener las carpetas organizadas con nombres claros y una estructura lógica reduce el tiempo de búsqueda, evita duplicados, facilita las copias de seguridad y mejora la productividad general del trabajo diario.

Herramienta central

Explorador de Windows

El Explorador de Windows (también conocido como Explorador de archivos) es la aplicación que permite navegar, visualizar y gestionar todos los archivos y carpetas del equipo. Es la herramienta fundamental para interactuar con el sistema de archivos.

Barra de direcciones

Muestra la ruta actual (ubicación) en la que te encontrás. Permite escribir directamente una ruta para acceder rápidamente a cualquier carpeta del sistema.

Ejemplo: C:\Users\Usuario\Documents\Facultad

Panel de navegación

Ubicado a la izquierda, muestra las carpetas principales y unidades de almacenamiento. Permite moverse rápidamente entre distintas ubicaciones sin escribir rutas.

Panel de contenido

Área principal que muestra los archivos y carpetas contenidos en la ubicación actual. Permite visualizar íconos, listas, detalles o miniaturas según la preferencia del usuario.

Cinta de opciones

Barra superior con botones y opciones organizadas por pestañas (Archivo, Inicio, Compartir, Ver). Facilita el acceso a las operaciones más comunes.

Barra de búsqueda

Ubicada arriba a la derecha, permite buscar archivos y carpetas dentro de la ubicación actual o en todo el equipo escribiendo palabras clave o nombres.

Acceso rápido

Sección especial que muestra las carpetas y archivos usados recientemente, facilitando el acceso a lo más frecuente sin navegar por toda la estructura.

Consejo

Podés abrir el Explorador de Windows de varias formas: haciendo clic en el ícono de carpeta en la barra de tareas, presionando las teclas Windows + E, o haciendo clic derecho en el botón Inicio y seleccionando "Explorador de archivos".

Tareas esenciales

Operaciones básicas con archivos y carpetas

Trabajar con archivos implica realizar operaciones cotidianas que permiten crear, mover, copiar, renombrar y eliminar información. Dominar estas acciones es fundamental para mantener el orden y la productividad.

Crear

Permite generar nuevos archivos o carpetas. Se puede hacer desde el menú contextual (clic derecho) o desde la cinta de opciones del Explorador.

Pasos:

  1. Clic derecho en el espacio vacío
  2. Seleccionar "Nuevo"
  3. Elegir "Carpeta" o el tipo de archivo
  4. Escribir el nombre y presionar Enter

Copiar y Pegar

Crea una copia exacta del archivo o carpeta en otra ubicación, manteniendo el original intacto. Es útil para duplicar información sin moverla de su lugar.

Pasos:

  1. Seleccionar el archivo o carpeta
  2. Clic derecho → "Copiar" o Ctrl + C
  3. Navegar a la ubicación destino
  4. Clic derecho → "Pegar" o Ctrl + V

Cortar y Pegar

Mueve el archivo o carpeta desde su ubicación original hacia otra. A diferencia de copiar, el elemento desaparece de su lugar inicial y aparece en el nuevo.

Pasos:

  1. Seleccionar el archivo o carpeta
  2. Clic derecho → "Cortar" o Ctrl + X
  3. Navegar a la ubicación destino
  4. Clic derecho → "Pegar" o Ctrl + V

Renombrar

Permite cambiar el nombre de un archivo o carpeta sin alterar su contenido ni su ubicación. Útil para corregir errores o reorganizar la información.

Pasos:

  1. Seleccionar el archivo o carpeta
  2. Clic derecho → "Cambiar nombre" o F2
  3. Escribir el nuevo nombre
  4. Presionar Enter

Eliminar

Envía el archivo o carpeta a la Papelera de reciclaje. Desde allí puede recuperarse si fue eliminado por error, o eliminarse definitivamente.

Pasos:

  1. Seleccionar el archivo o carpeta
  2. Presionar la tecla Suprimir o Delete
  3. Confirmar la eliminación
  4. (Se envía a la Papelera de reciclaje)

Seleccionar múltiples

Permite aplicar una operación (copiar, mover, eliminar) a varios archivos o carpetas al mismo tiempo, ahorrando tiempo y esfuerzo.

Métodos:

  • Ctrl + clic: selecciona elementos sueltos
  • Shift + clic: selecciona un rango continuo
  • Ctrl + A: selecciona todos los elementos

Atajo útil

Arrastrar y soltar

Podés mover archivos y carpetas arrastrándolos con el mouse desde su ubicación actual hacia la carpeta destino. Si mantenés presionada la tecla Ctrl mientras arrastrás, se crea una copia en lugar de moverlo.

  • Arrastrar solo: mueve el elemento
  • Ctrl + arrastrar: copia el elemento

Cuidado

Recuperar archivos eliminados

Cuando eliminás un archivo, va primero a la Papelera de reciclaje. Para recuperarlo, abrí la Papelera, seleccioná el archivo y hacé clic en "Restaurar". Si vaciás la Papelera, la recuperación es mucho más difícil o imposible.

Encontrar información

Búsqueda de archivos y carpetas

A medida que aumenta la cantidad de archivos en la computadora, encontrar uno específico puede volverse complicado. Windows ofrece herramientas de búsqueda que permiten localizar archivos rápidamente por nombre, tipo, fecha o contenido.

1. Búsqueda en el Explorador

El cuadro de búsqueda ubicado en la esquina superior derecha del Explorador permite buscar dentro de la carpeta actual. Basta con escribir parte del nombre del archivo.

Ejemplo: Si escribís "tarea" en la búsqueda, Windows mostrará todos los archivos que contengan esa palabra en su nombre.

2. Búsqueda desde el menú Inicio

Haciendo clic en el botón Inicio y escribiendo directamente, Windows busca en todo el equipo: archivos, aplicaciones, configuraciones y contenido web.

Ejemplo: Escribir "presupuesto" en el menú Inicio busca ese archivo en todas las carpetas del usuario.

3. Filtros de búsqueda

Windows permite refinar las búsquedas por tipo de archivo, fecha de modificación, tamaño o ubicación. Esto acelera la localización cuando hay muchos resultados.

  • Por extensión: tipo:.pdf
  • Por fecha: modificado:ayer
  • Por tamaño: tamaño:>10MB
Consejo

Si no recordás el nombre exacto del archivo, probá buscar palabras clave relacionadas con su contenido. Windows indexa el contenido de muchos tipos de archivos para facilitar la búsqueda.

Buenas prácticas

Organización efectiva de archivos

Una buena organización no surge por casualidad: requiere planificar una estructura lógica, usar nombres descriptivos y mantener el orden a lo largo del tiempo. Esto facilita el trabajo individual y colaborativo.

Buenas prácticas

Organización efectiva

  • Estructura jerárquica clara: Creá carpetas principales por tema o proyecto, y dentro de ellas subcarpetas más específicas.
  • Nombres descriptivos: Usá nombres que describan claramente el contenido. Evitá "documento1" o "nueva carpeta".
  • Consistencia en nombres: Establecé un criterio y mantenelo. Ejemplo: 2026-04-15_Reunion_Notas.docx

Estructura recomendada:
Documentos → Facultad → Informática → Trabajos Prácticos
Documentos → Facultad → Informática → Apuntes

Mantenimiento

Cuidado y respaldo

  • Evitar duplicados: Revisá periódicamente para eliminar archivos duplicados o versiones antiguas innecesarias.
  • Copias de seguridad: Realizá backups regulares de tus archivos importantes en un disco externo o en la nube (OneDrive, Google Drive).
  • Limpieza periódica: Dedicá tiempo cada cierto período para revisar y organizar. Eliminá lo que ya no sirve y archivá lo antiguo.
"Un minuto dedicado a organizar puede ahorrarte horas de búsqueda en el futuro."

Concepto técnico

Rutas de archivos y unidades

Cada archivo en la computadora tiene una dirección única llamada "ruta" que indica exactamente dónde está ubicado. Comprender las rutas ayuda a navegar el sistema de archivos con mayor precisión.

Estructura

Partes de una ruta

Una ruta de archivo tiene tres componentes principales: la unidad (disco), las carpetas intermedias y el nombre del archivo con su extensión.

Ejemplo de ruta completa:
C:\Users\Nacho\Documents\Facultad\Informatica\Clase3.pdf

  • C:\ → Unidad de disco principal
  • Users\Nacho\Documents\ → Carpetas intermedias
  • Clase3.pdf → Nombre del archivo

Unidades

Letras de unidad

Windows asigna letras a cada unidad de almacenamiento: C: para el disco principal, D:, E:, F: para otros discos, pendrives o unidades de red.

  • C:\ → Disco duro principal
  • D:\ → Unidad de DVD o segundo disco
  • E:, F:, G: → Pendrives, discos externos
Nota

La barra invertida (\) separa las carpetas en Windows. En otros sistemas operativos como Linux o macOS se usa la barra normal (/). Conocer la estructura de rutas es útil cuando trabajás con comandos o configuraciones avanzadas.
