# Landing de clase

Se creó una landing visual para la **Unidad 1 · Clase 1** en:

- `index.html`
- `styles.css`

## Cómo verla

Abrí `index.html` directamente en el navegador o levantá un servidor local, por ejemplo:

```powershell
python -m http.server 5500
```

Luego abrí `http://127.0.0.1:5500/`.
